Let's see how to build our own docker image.

# Creat and configure Dockerfile

```bash
$ sudo vim Dockerfile
$ cat Dockerfile
\#This is a sample Image 
FROM ubuntu 
MAINTAINER leopold.sga@outlook.com 

RUN apt-get update 
RUN apt-get install -y nginx
CMD [“echo”,”Image created”]
```

- `#` is followed by commments
- `FROM` represents what base image your image is based from
- `RNU` to run commands in your image
- `CMD` to display messages to user.

<!-- more -->

# Build image

## Used command

```bash
$ docker build  -t ImageName:TagName dir
```

- `-t` to maintain a tag to the image
